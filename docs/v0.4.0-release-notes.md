# CarryPigeon Desktop v0.4.0 发布说明

发布日期：2026-07-02

## 本次发布概述

v0.4.0 是一次以“功能体验 + 性能 + 可维护性”为核心的综合迭代。新增聊天内代码片段审查 MVP、诊断面板、启动反馈等面向普通用户与开发者的功能，同时在后端限流/缓存、前端内存与 IO、构建体积等方面做了大量优化。

## 主要亮点

### 新功能
- **聊天内代码片段审查**：本地 MVP，支持 triple-backtick 代码块行号渲染、点击行添加/删除注释、localStorage 持久化。
- **诊断面板**：设置页新增诊断弹窗，展示内存快照、趋势、状态、手动清理与最近应用日志。
- **启动反馈增强**：`StartupShell` 按阶段显示初始化文案（中英双语）。
- **Tauri 日志读取命令**：`read_app_log_lines` 支持诊断面板读取日志。

### 性能优化
- **前端消息列表内存窗口**：单频道保留最近 3000 条消息，限制长时间运行内存增长。
- **后端限流与缓存**：下载进度事件节流、TCP 状态事件去重、`get_config_value` TTL 缓存、高频配置写入批量 flush、聊天缓存 LRU 淘汰。
- **文件 IO 流式化**：分块读取本地文件，上传直接传递 `File`/`Blob`。
- **Web Worker 自适应消息排序**：大数据量 offload 到 Worker。
- **Composer 链接预览防抖**：400ms 防抖减少 IPC。

### 体验与可访问性
- i18n/ARIA 补缺：图标按钮添加 `aria-label` 与缺失 i18n 键值。
- 语音通话超时取消（Dialing/Ringing 60s）。
- 视频通话、屏幕共享、设备热插拔、降噪/静音状态同步。

### 代码质量
- Mock 配置收敛到 `@/shared/config/runtime`。
- Tauri 命令名统一收敛到 `TAURI_COMMANDS`。
- Rust 生产代码禁用 `unwrap`/`expect`/`panic!`/`todo!`。
- `.cargo/audit.toml` 记录已接受的 RUSTSEC-2023-0071 风险（项目仅使用 SQLite）。
