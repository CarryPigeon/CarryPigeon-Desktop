## 2. 建议新增组件（为 PRD 补齐）

插件中心：
- `PluginCenterPage.vue`（列表视图）
- `PluginDetailPage.vue`（详情视图）
- `PluginCard.vue`（卡片）
- `PluginInstallProgress.vue`（下载/校验/解压进度）
- `PluginRequiredBadge.vue`（required 徽章/邮戳风格）

required 向导：
- `RequiredSetupPage.vue`（缺失列表 + 跳转）
- `RequiredChecklist.vue`（missing_plugins 状态：缺失/安装/启用/失败）

未知 domain 消息降级：
- `UnknownDomainBubble.vue`（preview/引导安装/错误说明）

连接状态条：
- `ConnectionStatusPill.vue`（弱提示，放顶栏右侧）

## 3. 组件状态（最少要覆盖）

每个“可操作组件”至少要有：
- `idle` / `loading` / `error` / `disabled`
- 失败原因可展开（尤其插件 enable/verify）
- 可恢复动作（重试/回滚/换源）
