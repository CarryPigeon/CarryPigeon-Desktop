# 项目简介

CarryPigeon 的桌面端（Tauri + Rust + Vue 3）。

## 技术栈

- 框架：Tauri
- 后端：Rust
- 前端：Vue 3

## 本地运行

```shell
pnpm run tauri dev
```

## 打包

```shell
pnpm tauri build
pnpm tauri build --no-bundle
```
